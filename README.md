This document describes the methodology governing CCIR's published reference rates for GPU compute. It covers eligibility criteria, computation procedures, confidence flag definitions, the spread framework, and source taxonomy. All design decisions are documented with the rationale required to assess fitness for use in credit documents, collateral appraisals, and covenant frameworks.
CCIR holds no financial interest in any GPU marketplace from which it collects data. This structural independence is a prerequisite for use of CCIR rates as reference instruments in lending agreements. The methodology reflects that position: every filter and computation choice is made to maximize representativeness and auditability, not to favor any particular market participant or price level.
